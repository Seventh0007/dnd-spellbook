## 1. 项目概述

### 1.1 项目背景
创建一个面向中文桌游（跑团）玩家的D&D法术查询工具网站，采用**D&D官方Stat Block风格**设计。

### 1.2 目标用户
- D&D 5版玩家（DM和PC）
- 跑团新手和资深玩家
- 需要快速查询法术规则的用户

### 1.3 核心目标
- 提供快速、便捷的法术查询功能
- 支持多版本规则（5e经典版、2024修订版）
- 提供完整的学习路径（新手→进阶→熟练）
- **采用沉浸式D&D Stat Block视觉风格**

## 4. 设计需求 - D&D Stat Block风格

### 4.1 核心视觉风格：D&D Stat Block

#### 4.1.1 整体风格定位
**完全模仿D&D官方规则书中的Stat Block设计**，包括：
- 怪物Stat Block
- 法术Stat Block
- 物品Stat Block

**参考资料：**
- D&D 5e Player's Handbook 法术章节
- D&D 5e Monster Manual 怪物Stat Block
- D&D Beyond 网站的法术展示风格

#### 4.1.2 关键设计元素

**1. 仿古纸张背景（Parchment Background）**
- 颜色：#fdf1dc（经典D&D羊皮纸色）
- 纹理：subtle parchment texture（可以使用CSS图案或轻量图片）
- 替代方案：纯色 #fdf1dc 或 #faf3e0

**2. 装饰性边框（Ornamental Border）**
- 边框样式：double line（双线边框）
- 颜色：#7a200d（深红色，D&D经典红）
- 边角装饰：CSS伪元素实现简单的装饰性边角

**3. 分隔符（Dividers）**
- 样式：double line（双线分隔符）
- 颜色：#7a200d（深红色）
- 粗细：2px double
- 用于：标题下方、元数据之间、内容区块之间

**4. 字体方案（Serif Fonts）**
- **英文标题**：Georgia, Garamond, "Noto Serif SC", serif
- **英文正文**：Georgia, Garamond, serif
- **中文标题**：Georgia, "Noto Serif SC", SimSun, serif
- **中文正文**："Noto Serif SC", SimSun, serif
- **备选方案**（如果特殊字体无法加载）：Georgia, "Times New Roman", serif

**5. 色彩方案（D&D Classic）**
- **主色调（纸张）**：#fdf1dc（羊皮纸色）
- **强调色（边框/标题）**：#7a200d（深红色，D&D经典红）
- **文本色（主文本）**：#222（深灰色，接近黑色但不刺眼）
- **文本色（次要文本）**：#666
- **文本色（引言）**：#888（灰色斜体）
- **链接色**：#7a200d（深红色，与标题一致）

**6. 法术环数颜色编码（可选）**
- 戏法：#7a200d（红）
- 1-3环：#c9ad6a（金）
- 4-6环：#0c4ca3（蓝）
- 7-9环：#6a0dad（紫）

### 4.4 排版规范（Typography）

- **正文（描述）**：1em, serif, #222, line-height: 1.6
- **小字（来源、页码）**：0.85em, serif, #999, right-aligned

- **标题下方间距**：20px（加双线分隔符）
- **段落行高**：1.6 - 1.8（易读性）
- **卡片内边距**：20px - 30px
- **元素间间距**：15px - 20px

### 4.5 交互设计（保持Stat Block风格）

#### 4.5.1 悬停效果
- **卡片悬停**：
  - 边框发亮：box-shadow: 0 0 15px rgba(122, 32, 13, 0.6)
  - 轻微上浮：translateY(-5px)
  - 过渡动画：all 0.3s ease
  
- **按钮悬停**：
  - 背景变亮：background: #f5e6c8（稍亮的羊皮纸色）
  - 边框发亮：box-shadow: 0 0 10px rgba(122, 32, 13, 0.6)

#### 4.5.2 动画效果
- **淡入淡出**：opacity 0→1，持续0.3s
- **滑动效果**：translateY(-10px)
- **弹窗动画**：fadeIn + slideUp（保持Stat Block风格，不使用现代感太强的动画）

- **样式**：**Stat Block风格**
  - 背景：#fdf1dc
  - 边框：double line, #7a200d（深红色）
  - 字体：衬线字体、#7a200d

## 9. 风险与注意事项

### 9.1 技术风险
- **法术数据完整性**：D&D法术数量庞大，收集完整数据需要大量时间
  - **缓解措施**：先完成常用法术（50-100个），后续逐步补充
  
- **浏览器兼容性**：某些高级特性（如Clipboard API、伪元素装饰）在旧浏览器不支持
  - **缓解措施**：提供降级方案（简化边框、无装饰性边角）

- **字体加载**：特殊字体（如Cinzel、Trajan Pro）可能加载慢或不支持中文字符
  - **缓解措施**：使用Google Fonts的替代方案，或仅对英文字符使用特殊字体

### 9.2 法律风险
- **版权问题**：D&D法术描述文本可能涉及版权
  - **缓解措施**：使用官方公开数据或自行编写描述
  
- **商标问题**：使用D&D logo、DUNGEONS & DRAGONS文字可能涉及商标
  - **缓解措施**：避免使用官方Logo，仅使用文字描述

### 9.3 用户体验风险
- **数据加载慢**：如果法术数据过大，可能影响加载速度
  - **缓解措施**：分页加载或虚拟滚动
  
- **搜索性能**：如果法术数量过多，搜索可能变慢
  - **缓解措施**：使用索引或Web Worker

- **移动端体验**：Stat Block风格可能在小屏幕上显示不佳
  - **缓解措施**：移动端简化布局，但保持Stat Block风格要素
